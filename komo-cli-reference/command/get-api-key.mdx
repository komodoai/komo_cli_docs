## get-api-key
The `get-api-key` command is used to log into Komodo and retrieve your API key. This key is required to use Komodo's services. To retrieve your API key, simply run the following command:

```
komo get-api-key
```

You will be prompted to enter your Komodo username (email) and password. Once authenticated, your API key will be displayed in the output.
